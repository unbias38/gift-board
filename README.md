# 股東會紀念品看板 gift-board

**2026 年台灣股東會紀念品查詢工具，每日自動更新。**

🔗 線上使用：**https://unbias38.github.io/gift-board/**

收錄 700 多檔上市、上櫃、興櫃公司的股東會紀念品資訊，專為零股族與紀念品收集者設計。

## 功能

- **今日看板**：今日最後買進日／代領截止／電子投票截止，一目了然不漏接
- **未來 3 日預警**：三種期限合併排序，提前準備
- **我的持股**：設定自選股（存在瀏覽器，不上傳），自動標星、補單提醒
- **全部資料查詢**：搜尋＋多條件篩選（年度／市場別／零股可代領／領取限制／票券類／eGift），點欄位排序
- **零股友善資訊**：1 股可代領名單、需證件正本／紙本通知單等領取限制
- **eGift 標示**：支援集保「紀念品電子化」的公司，電投完線上領、免排隊免補單
- **股價 CP 值**：每日更新股價，快速找出低成本高價值標的
- **統計圖表**：票券 vs 實物比例、每月場次分布

## 架構

```
資料來源（每日自動爬取）
  股代網 / 零股屋 / 集保 eGift 名單 / Yahoo Finance
        ↓
Google Apps Script（每日 07:00 排程）
        ↓
Google Sheets（資料庫）
        ↓ JSON API
GitHub Pages（本 repo，純靜態前端）
```

前端為單一 HTML 檔（Tailwind CSS + Chart.js），透過 GAS Web App API 取得資料；後端爬蟲與資料庫由 Google Apps Script + Google Sheets 承擔，全套零主機成本。
